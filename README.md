# 💫 About Me:
I am a Computer Science graduate from BML Munjal University with experience in AI Systems, Data Engineering, and GenAI applications. Currently I am working at Consilio where I build SQL data pipelines and AI-ready data infrastructure for LLM and RAG systems. Previously I worked as an AI App Developer Intern where I developed an Agentic AI application using FastAPI, Docker, and multi-agent workflows. My main expertise is in Python, LLM applications, RAG pipelines, vector databases, and scalable AI backend systems. I am particularly interested in AI infrastructure and building production-grade GenAI systems.
